dplyr 패키지의 %>% 연산자는 Ctrl + Shift + M으로 쉽게 입력하실 수 있습니다.

이 외에 select와 함께 유용하게 사용할 수 있는 함수는

  • start_with(“abc”): ’abc’로 시작하는 이름

  • end_with(“xyz”): ’xyz’로 끝나는 이름

  • contains(“ijk”): ’ijk’를 포함하는 이름

  • one_of(c(“a”, “b”, “c”)): 변수명이 a, b, c 중 하나

  • matches(“(.)\1”): 정규표현식 조건

  • num_range(“x”, 1:3): x1, x2, x3


mutate : 새로운 변수 정의

mutate는 기존에 있는 변수 외에 새로운 변수를 만들 때 사용하는 함수이다.

BMI와 같이 키와 몸무게가 있는 경우 식을 통해 새로운 변수를 만들 수 있다.

또는 연속형 자료로된 나이를 20대, 30대, 40대와 같은 연령대인 범주형자료로 바꿀 수 있다.

\[BMI=Weight/Height^2\]

BMI가 18.5 이하 : “저체중”, 18.5~23 : “정상”, 23~25 : “과체중”, 25~ : “비만” 이다.

mutate함수와 pre.weight를 이용하여 계산하자.

  • 기존 자료에 %>% 만 연결하면 수정된 자료가 만들어 지지 않는다.

  • 새로 생성된 변수가 반영된 데이터를 사용하기 위해선 <-를 이용해 새로운 이름으로 데이저를 명명하자.

ex1 <- ex %>% mutate(BMI.pre = pre.weight/(Height^2)*10000) # 키의 단위는 m이므로 곱하기 10000 
ex1 %>% mutate(BMI.cate = ifelse(BMI.pre<=18.5, "저체중",
                                 ifelse(BMI.pre<=23, "정상",
                                        ifelse(BMI.pre<=25, "과체중","비만")))) %>% head(10)

summarize : 집단요약

많은 수의 집단을 요약할 때 사용하는 함수로 대푯값, 산포도, 빈도가 주로 사용된다.

대푯값 : 평균, 중위수, 최빈값 산포도 : 분산, 표준편차, 범위, 등

summarize함수는 group_by와 함께 사용할때 이용가치가 높으며 기본 R에서는 aggregate 함수가 사용된다.

T검정, 분산분석과 함께 사용하면 좋다.

ex1 %>% filter(!is.na(gender)) %>% 
  group_by(gender) %>%
  summarize(count=n(), meanBMI=mean(BMI.pre), sdBMI=sd(BMI.pre)) 
## # A tibble: 2 × 4
##   gender count meanBMI sdBMI
##    <int> <int>   <dbl> <dbl>
## 1      0    43    24.2  3.81
## 2      1    33    26.1  3.87

만약 summarize_all을 사용하면 모든 변수에 적용된다.

ex %>% summarize_all(mean)
##   Person gender      Age   Height pre.weight     Diet weight6weeks
## 1   39.5     NA 39.15385 170.8205   72.52564 2.038462     68.68077

tc.data2<-t.data[t.data$date=="2017-01-01",]
coordinates(tc.data2) <-c("lon","lat")

grd <- as.data.frame(spsample(tc.data2, "regular", n=50000))
names(grd)<- c("lon","lat")
coordinates(grd) <- c("lon","lat")
gridded(grd)     <- TRUE  # Create SpatialPixel object
fullgrid(grd)    <- TRUE  # Create SpatialGrid object

tc.idw <- gstat::idw(avg ~ 1, tc.data2, newdata=grd, idp=2.0)
## [inverse distance weighted interpolation]
if(!require(raster)) install.packages("raster"); library(raster)

r <- raster(tc.idw)
plot(r)

if(!require(mapdata)) install.packages("mapdata"); library(mapdata)

map(database = "worldHires", region='South Korea', add=TRUE)
contour(r, add=T, lty=2)
